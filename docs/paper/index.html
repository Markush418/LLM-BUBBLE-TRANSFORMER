Bubble Transformer V5

Focus-Inspired Sinkhorn Token Grouping for Causal Language Model Attention
Marcus | July 2026
GATE PASSED (+0.16% PPL) 501 Tests Passing Open Source

Abstract

We present Bubble Transformer V5 (BT V5), a hybrid attention mechanism that uses Sinkhorn normalization for token grouping while preserving standard softmax within groups. Through single-layer frozen swap experiments on Qwen3-0.6B with WikiText-2, we find that FocusDeltaNet achieves PPL=22.550 (+0.16%) at Layer 7 with λ=0.3, passing the ≤2% PPL gate. Multi-layer experiments show FocusDeltaNet scales to 3 layers at +1.38% PPL. Needle-in-a-Haystack evaluation at 2K tokens shows 100% retrieval accuracy.

The Problem

Previous BT V5 variants tried replacing softmax with doubly-stochastic attention (SIRI). All failed the ≤2% PPL gate:

VariantPPL DriftGate
Pure SIRI+211%FAIL
Row-stochastic+30%FAIL
Hybrid approaches+2.39% to +2.85%FAIL

The core issue: doubly-stochastic normalization destroys the sharp attention distributions language models need.

The Insight

The Focus paper showed that Sinkhorn can improve PPL when used for token grouping rather than attention normalization.

S = Q @ K^T / sqrt(d)           # dot-product scores
S = S + psi                      # Power Diagram bias
groups = Sinkhorn(S, tau=1)      # soft grouping (doubly-stochastic)
attn = softmax(S + log(groups))  # softmax within groups (preserved peakedness)
output = attn @ V

Best Result: FocusDeltaNet L7 (λ=0.3)

PPL = 22.550 (+0.16%) — 12.5x better than the 2% gate threshold

Optimal configuration: ε=0.001, τ=1, λ=0.3 (30% DeltaNet, 70% Focus)

Honest Caveats

  • Single-layer swap only (not validated with pretrain from scratch)
  • Limited long-context (NIAH at 2K tokens, GTX 1650 VRAM limit)
  • No downstream evaluation (MMLU, HumanEval, GSM8K)
  • Power Diagram ψ absorbed by Sinkhorn column normalization
  • L9 anomaly unexplained (+2.28% PPL while neighbors pass)
